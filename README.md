# K72 - Modern Creative Agency Portfolio

![K72 Agency](https://k72.ca/uploads/teamMembers/Carl_480x640-480x640.jpg)

A high-performance, visually stunning creative agency portfolio built with **React**, **Vite**, and **GSAP**. This project showcases modern web design principles, including smooth scroll-driven animations, video backgrounds, and a clean, minimalist UI/UX.

## ✨ Features

- **🚀 GSAP Animations**: Leverages GSAP ScrollTrigger for smooth, high-fidelity animations and interactive elements.
- **🗺️ Interactive Navigation**: Features a dynamic Navbar and a full-screen navigation overlay for a clean, non-intrusive UI.
- **🎥 Immersive Hero Section**: Features full-screen video backgrounds with overlaid typography for a cinematic feel.
- **👥 Dynamic Team Section**: Interactive image sequences that change dynamically as you scroll through the agency's story.
- **🎨 Modern UI/UX**: Built with a "less is more" approach, focusing on high-quality assets and elegant typography.
- **📱 Responsive Design**: Fully optimized for various screen sizes, ensuring a premium experience on both desktop and mobile.
- **⚡ Fast Performance**: Powered by Vite for near-instant development and optimized production builds.

## 🛠️ Tech Stack

- **Frontend**: [React](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [TailwindCSS](https://tailwindcss.com/)
- **Animations**: [GSAP](https://greensock.com/gsap/) (ScrollTrigger, @gsap/react)
- **Routing**: [React Router](https://reactrouter.com/)

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/k72-portfolio.git
   ```

2. Navigate to the project directory:
   ```bash
   cd k72-portfolio
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the development server:
```bash
npm run dev
```

### Production

Build the project for production:
```bash
npm run build
```

Preview the production build:
```bash
npm run preview
```
